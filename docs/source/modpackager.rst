CollabXR Mod Packager
====================

The `com.ecmodding.modpackager`` package is used to bundle and upload mods to a mod repository for use within CollabXR.

It is recommended you have some understanding of Unity's `Asset Bundles`_ system before using the Mod Packager.

.. _Asset Bundles: https://docs.unity3d.com/6000.0/Documentation/Manual/AssetBundlesIntro.html
