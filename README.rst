Template for the Read the Docs tutorial
=======================================

======
Development
======

1. Install pre-commit (either globally or in virtual environment): ``pip install pre-commit``
2. Install Prettier for code formatting: ``npm i -g prettier``
3. Setup pre-commit hooks for code cleanliness: ``pre-commit install``

This GitHub template includes fictional Python library
with some basic Sphinx docs.

Read the tutorial here:

https://docs.readthedocs.io/en/stable/tutorial/
